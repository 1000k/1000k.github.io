git の submodule を最新にする方法

「そんなの git submodule update で一発だろ」と思っていましたが、全然違いました。このコマンドをいくら叩いても、標準出力には何も表示されず、submodule はちっとも更新されません。

submodule を最新の状態にするコマンドは git submodule foreach git pull origin master です。
